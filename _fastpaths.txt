# Fast URL patterns (html-preview) / 10k tokens/read helper
#
# Use this when you already know the ID(s) and want to jump straight to the text.

## Statutes (ai-law-db)
- Text: `/html-preview/ai-law-db/text/{law_code}/{article}.txt`
- Cite (optional): `/html-preview/ai-law-db/enhanced/{law_code}/{article}.html#pX`

## Circulars / Basic circulars (ai-tsutatsu-db)
- Text: `/html-preview/ai-tsutatsu-db/text/{doc_code}/{item_id}.txt`
- Cite (optional): `/html-preview/ai-tsutatsu-db/enhanced/{doc_code}/{item_id}.html#pX`
- Common doc_code:
  - `hojinzei_kihon_tsutatsu`（法人税基本通達）
  - `shohizei_kihon_tsutatsu`（消費税基本通達）

## Cases / Rulings (ai-hanketsu-db)
- Start: `/html-preview/ai-hanketsu-db/quickstart.txt`
- Corporate tax (example): `/html-preview/ai-hanketsu-db/houjinzei/core/{id}.txt`

## Papers / Guidelines (ai-paper-db) [packs/shards]
- Corpus quickstart:
  - `/html-preview/ai-paper-db/oecd-tpg-2022/quickstart.txt`
  - `/html-preview/ai-paper-db/oecd-beps/quickstart.txt`
  - `/html-preview/ai-paper-db/nta-tp-audit/quickstart.txt`
- Text: `/html-preview/ai-paper-db/{corpus}/packs/{pack_file}.txt`

## Tax treaties (ai-treaty-db)
- Corpus quickstart: `/html-preview/ai-treaty-db/jp-tax-treaties/quickstart.txt`
- Text (pack): `/html-preview/ai-treaty-db/jp-tax-treaties/packs/{doc_id}-pack-XX.txt`
- Fulltext (long; not recommended for mobile-AI): `/html-preview/ai-treaty-db/jp-tax-treaties/source/text/{doc_id}.txt`
- Source PDF: `/html-preview/ai-treaty-db/jp-tax-treaties/source/raw/{doc_id}.pdf`
